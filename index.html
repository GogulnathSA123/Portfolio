02. About Me

I'm an MSc Robotics student with a deep passion for building intelligent systems that can perceive, reason, and act in the real world. My research focus lies at the intersection of Vision-Language-Action (VLA) models and embodied AI — working toward robots that understand both visual scenes and natural language to execute meaningful physical tasks.

My journey spans from low-level embedded systems and sensor fusion on Arduino-based robots to high-level machine learning pipelines for computer vision and reinforcement learning. I enjoy the full stack of robotics — from soldering encoders to training neural networks.

When I'm not in the lab, I'm exploring the latest research in foundation models for robotics, sim-to-real transfer, and multimodal perception systems. I believe the next frontier in AI is agents that don't just think — they act.

03. Featured Projects

Bayesian Leader-Follower Robot

Embedded robotics system where a follower robot autonomously tracks a leader using Bayesian sensor fusion. Replaces raw IR thresholding with a probabilistic pipeline — Welford's online calibration, Gaussian log-likelihoods, log-sum-exp normalization, and MAP decision-making with confidence thresholds.

C++ Arduino Bayesian Filter Sensor Fusion Pololu 3Pi+ Embedded
View Code

Apple Detection with SVM

Hybrid image processing + machine learning pipeline for automated apple detection in orchard images. Uses HSV color thresholding for candidate generation, morphological filtering, HOG feature extraction, and a linear SVM classifier with three-tier confidence scoring. Trained on Fuji-SfM and MinneApple datasets.

Python OpenCV HOG SVM Scikit-learn Computer Vision
View Code

TurtleBot3 SLAM & Navigation

Full ROS1 catkin workspace implementing real-time 2D SLAM using LiDAR-based Gmapping (Rao-Blackwellized Particle Filter). Includes differential drive control, Kalman filter localization, TF frame integration, joystick teleoperation, camera/LiDAR perception, and occupancy grid mapping in custom Gazebo environments.

ROS Gazebo SLAM Kalman Filter LiDAR C++ Python
